Ordered List of HTML Tags

  1. <html>
    He root element of an HTML document.
  2. <head>
    Contains meta-information about the HTML document, such as the title.
  3. <title>
    Sets the title of the HTML document displayed in the browser.
  4. <body>
    Contains the content of the HTML document displayed in the browser.
  5. <h1>
    Defines a top-level heading.
  6. <ol>
    Creates an ordered list.
  7. <li>
    Represents a list item within an ordered or unordered list.
  8. <Mark>
    Define its highlighted text.
  9. <ul>
    Creates an unordered List.
  10. <p>
    Its Define Paragraph.
